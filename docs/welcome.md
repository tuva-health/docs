## Why Use Tuva?

Tuva transforms your healthcare data into an analytics-ready data model that enables you to answer critical healthcare analytics questions rapidly.  There are two main reasons why people use Tuva:

1. You need to run analytics across multiple healthcare data sources which come in multiple formats --> **doing this requires a common, analytics-ready data model, which Tuva provides.**

2. You want to run all data transformation inside your cloud data warehouse (e.g. Snowflake, Databricks) using [dbt](https://www.getdbt.com/) so you have 100% control and transparency into data transformation --> **this is exactly how Tuva works.**

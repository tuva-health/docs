Tuva notebooks provide practical, analysis-first examples you can run on top of the Tuva Core Data Model and Data Marts.

- [Tuva Notebooks Repository](https://github.com/tuva-health/tuva-notebooks)

Common use cases include:

- exploratory data analysis
- cohort building
- quality and cost analysis workflows
- model prototyping and experimentation
